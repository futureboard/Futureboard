# FutureBoard Studio

A modern DAW experiment built around a fast editor-like workflow.

This repository currently contains the web/electron prototype used to explore:
- arrangement editing
- mixer workflow
- MIDI editing
- command palette driven UX
- future native audio/runtime architecture

The current version is a prototype.  
The long-term plan is to move the audio/runtime layer into a native engine.

## Structure

```txt
apps/web       Web prototype
apps/electron  Electron desktop shell
core           Native/audio core experiments
````

## Status

Experimental. Expect chaos.
