# Futureboard Studio — User Manual

Welcome to **Futureboard Studio**, the native desktop digital audio workstation
in the Futureboard family. This manual covers the Studio surface: recording,
arranging, mixing, plugin hosting, automation, and export.

> Futureboard ships in several surfaces — **Express** (web), **Lite**
> (Electron), and **Studio** (native Rust/GPUI). This manual documents
> **Studio**. Most concepts carry over to the other surfaces.

## Quick reference

| Task | Where |
|------|-------|
| Create / open a project | **File** menu, or the Welcome window |
| Add a track | **Track ▸ Add**, or the timeline context menu |
| Show / hide panels | **View** menu |
| Open the mixer | **View ▸ Mixer** |
| Insert a plugin | Track inspector or mixer channel strip |
| Export audio | **File ▸ Export Arrangement** |
| Keyboard shortcuts | **Help ▸ Keyboard Shortcuts** (Ctrl+/) |
| This manual | **Help ▸ Documentation** (F1) |

## Getting help

- **Help ▸ Documentation** (F1) opens this manual.
- **Help ▸ Keyboard Shortcuts** (Ctrl+/) opens the shortcut editor and keymap picker.
- **Help ▸ Diagnostics** opens performance and system diagnostics.
- **Help ▸ Report an Issue** links to the issue tracker.
