# The Interface

The Studio workspace is a compact, dense desktop layout built for long
sessions. It is organized into a menu bar, a transport, dockable panels, and
the central arrangement.

## Menu bar

Along the top: **File**, **Edit**, **View**, **Transport**, **Track**,
**Window**, **Help**, and more. On macOS the native menu bar is used; on
Windows and Linux the menu bar is drawn in-window and collapses to a compact
control on narrow widths.

## Transport

The transport bar controls playback and recording:

- **Play / Stop** — start and stop the timeline.
- **Record** — arm and capture into armed tracks.
- **Loop / Cycle** — repeat a region.
- Tempo, time signature, and playhead position readouts.

## Panels

Panels dock around the central arrangement and are toggled from the **View**
menu:

- **Browser** (left) — collections, library, and places for files, plugins, and presets.
- **Inspector** (right) — properties for the selected track or clip.
- **Mixer** (bottom or floating) — channel strips, inserts, sends, meters.
- **Editor** (bottom) — the MIDI piano roll and related editors.

Panels can be shown, hidden, and resized. DAW-critical surfaces use explicit
measured layout so rulers, lanes, and the playhead stay aligned across resizes,
zoom, and DPI changes.

## The arrangement

The central arrangement shows tracks as horizontal lanes against a time ruler:

- **Track headers** (left) — name, mute/solo/arm, height, color.
- **Timeline ruler** (top) — bars, beats, and time.
- **Lanes** — clips, automation, and the moving playhead.

Scroll horizontally to move through time, vertically through tracks, and zoom
to change the time scale.

## Floating windows

Some tools open as separate windows so you can place them on a second monitor:

- **Mixer** (floating)
- **Audio Connections** (routing matrix) — **Audio** menu
- **Plugin editors** — hosted native plugin UIs
- **Chord / Lyric displays**

## Command palette

A command palette gives quick keyboard access to actions without hunting
through menus. Combined with the [keyboard shortcuts](./keyboard-shortcuts.mdx),
it keeps common operations at your fingertips.
