# Keyboard Shortcuts

## The shortcut editor

Open **Help ▸ Keyboard Shortcuts** (Ctrl+/) to view and edit every binding and
to switch keymap profiles. This window is the **source of truth** for the
active bindings — it always reflects your current profile and any custom
overrides.

## Keymap profiles

Studio ships with profiles modeled on popular DAWs so your muscle memory
carries over:

- **Futureboard** (default)
- **Ableton**
- **FL Studio**
- **Pro Tools**
- **Studio One**
- **Cubase**

Pick a profile in the shortcut editor. Bindings shown in menus update to match.

## Focus priority

Shortcuts respect where your focus is, so the same key does the right thing in
context. Priority, highest first:

1. Modal dialog
2. Text input / search / numeric / color field
3. MIDI editor
4. Timeline
5. App / global

This is why edit keys (Ctrl/Cmd+A/C/V/X, Delete) act on clips or notes in the
timeline and piano roll, but never fire while you are typing in a text field.

## Common actions

Exact keys depend on the active profile — check the shortcut editor for your
bindings. Common actions include:

| Action | Where it applies |
|--------|------------------|
| Play / Stop | Transport |
| Record | Transport |
| Select all | Timeline / MIDI editor |
| Copy / Cut / Paste | Timeline / MIDI editor |
| Delete | Timeline / MIDI editor / automation |
| Undo / Redo | Global |
| Documentation | Global (F1) |
| Keyboard Shortcuts | Global (Ctrl+/) |

## Customizing

Rebind actions in the shortcut editor. Overrides are saved with your keymap
selection and persist across sessions.
